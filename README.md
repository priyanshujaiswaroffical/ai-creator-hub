# 🚀 AI-Native Creator Hub

A premium, full-stack startup + portfolio platform blending **Cinematic 3D Web Development**, **Autonomous AI Engineering**, and **Cinematic Production**.

![Tech Stack](https://img.shields.io/badge/Stack-Next.js%2016%20%7C%20FastAPI%20%7C%20Three.js%20%7C%20Gemini-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## ✨ Features

- **🎨 Immersive 3D Hero:** A high-performance, interactive geometric scene built with React Three Fiber and Shaders.
- **🤖 Autonomous AI Chat:** Integrated Google Gemini-powered chatbot with RAG (Retrieval-Augmented Generation) for intelligent portfolio interaction.
- **🌓 Advanced Theme Engine:** Seamless Light/Dark mode transitions with zero-flicker performance.
- **📱 Responsive Excellence:** Mobile-first, liquid layouts using Tailwind CSS v4.
- **⚡ Fast-API Backend:** High-speed Python backend handling AI logic, lead capture, and notifications.

## 🛠️ Tech Stack

### Frontend
- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4 + Framer Motion
- **3D Engine:** Three.js + React Three Fiber + Drei
- **State:** Zustand

### Backend
- **Framework:** FastAPI (Python 3.12+)
- **AI Engine:** Google Gemini Pro
- **Database:** Supabase (PostgreSQL)
- **Validation:** Pydantic v2

## 📦 Installation

### 1. Clone the Repository
```bash
git clone https://github.com/priyanshujaiswaroffical/ai-creator-hub.git
cd ai-creator-hub
```

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### 3. Backend Setup
```bash
cd ../backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
