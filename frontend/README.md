# 🎨 Creator Hub — Frontend

The immersive UI layer of the AI-Native Creator Hub. Built with Next.js 16, Tailwind 4, and React Three Fiber.

## 🚀 Key Features
- **3D Canvas:** Interactive icosahedron distortion with R3F.
- **Glassmorphism UI:** Ultra-modern aesthetic using Tailwind 4's new engine.
- **Theme Sync:** LocalStorage persistent dark/light mode.
- **Responsive Layout:** Adaptive sections for all devices.

## 🛠️ Development
```bash
npm install
npm run dev
```

## 🏗️ Build
```bash
npm run build
```
The build output will be located in the `/out` folder, which is automatically served by the FastAPI backend.
